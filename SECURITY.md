# Security Policy

## Supported Versions

The table below outlines which versions of DirFlow are currently receiving security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public GitHub issue. 

Instead, please report it privately:
1. Send an email to **coolguynova0302@gmail.com**.
2. Provide a detailed summary of the vulnerability, including steps to reproduce and any potential impact.
3. You will receive an initial response within **48 hours** confirming receipt.
4. If the vulnerability is accepted, we will coordinate a fix and release a patch update.
